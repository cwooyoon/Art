대전시립미술관이 준비한 올해 과학예술강연은 인공지능(AI)를 키워드로 기획된 수업으로, 인공지능과 함께 커가는 어린이를 대상으로 진행하는 쉽고 재미있는 강연입니다.

5명의 외부 전문가를 초청하여 진행하는 이번 강연은 나날이 발전하고 있는 과학(로봇) 기술의 다양한 방면을 어린이가 생생하고 풍부하게 접할 수 있도록 도와줍니다.

미술관에서 과학자의 창의적인 상상력을 만나보는 과학예술강연에 많은 관심과 신청 바랍니다.
